# Data-Example


Here are some photos for 3 signs classification (Right, Left, and Straight).

This is meant to be a helpful structure for [RobotFramework-ImageDetection](https://github.com/Alpha-Centauri-00/robotframework-imagedetection) library.

    Data
        ├── training
        │   ├── right
        │   │   ├── photo1.png
        │   │   ├── photo2.png
        │   │   └── photo2.png
        │   ├── left
        │   │   ├── photo1.png
        │   │   ├── photo2.png
        │   │   └── photo2.png
        │   ├── Straight
        │   │   ├── photo1.png
        │   │   ├── photo2.png
        │   │   └── photo2.png
        ├── validation
        │   ├── right
        │   │   ├── new_photo1.png
        │   │   ├── new_photo2.png
        │   │   └── new_photo2.png
        │   ├── left
        │   │   ├── new_photo1.png
        │   │   ├── new_photo2.png
        │   │   └── new_photo2.png
        │   ├── Straight
        │   │   ├── new_photo1.png
        │   │   ├── new_photo2.png
        │   │   └── new_photo2.png



> ⚠️ Make sure the main dataset folder is named "Data," with two subfolders inside it called "training" and "validation." Each of these subfolders should contain the photos.
